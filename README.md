# Vi2i0n
## **The Visual Acuity App**

A Tkinter app that helps user determine their visual acuity.


## **Technologies:**
	Tkinter
	opencv2


## **LocalSetup:**
	-Install All Dependencies
	-pip3 install -r requirements.txt


## **Run the File:**
	python3 Vision_Acuity_App.py
